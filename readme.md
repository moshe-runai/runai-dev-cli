# CLI for the runai developers
## Features:
1. Port forwart to:
```
  1. Traefik
  2. Backend
  3. Database
  4. Tenants Manager
  5. Thanos
```
2. Edit Deployment:
```
  1. Backend
  2. Frontend
  3. Asset Service
  4. Policy Service
```
3. Activate Kube config
4. List Environments
4. Edit backend configmaps
4. Rollout backend

## Installation
```
npm i -g .
```

## Use:
```
runai-dev
```
